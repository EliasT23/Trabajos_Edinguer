### Mascotas
- Este programa utiliza un if y elif para identificar si tu mascota es un perro o gato
```python
nombre = input("Ingresa el tipo de mascota  que tienes: \n")
if "perro" in nombre:
    print("Es un perro")
elif "gato" in nombre:
    print("Es un gato")
else:
    print("Mascota desconocida")
